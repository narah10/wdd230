Typography

I will be using only two fonts for this website

  • Crimson Text: This font style is used for the all different headings. (h1,h2,h3) I chose to use this font because it is a serif font and this will help headings pop out more in the background. It will create a form of hierarchy as viewers look through the website.
  • Work Sans: This font will be used for all paragraphs and other texts for the nav section. This will also create contrast in the fonts because it is a sans-serif font.
